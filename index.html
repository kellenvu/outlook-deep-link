Deep Link Creator for Outlook Emails

How to Use This App

  1. Go to your Outlook inbox.
  2. Open the Chrome DevTools Console (press F12).
  3. Go to the Network tab.
  4. In the search bar, search for service.svc?action=GetItem.
  5. Click the email you want to get a deep link for.
  6. A new item should appear in the network traffic. Click it, then click Response.
  7. Copy and paste the entire response into the text input above.
  8. The app automatically generates a deep link to the email, which you can copy/paste elsewhere.
