Deep Link Creator for Outlook Emails

What Is a Deep Link?

A deep link is a hyperlink that directs you to a specific location within a website, instead of the landing page. For example, a deep link to an Outlook email will take you to a specific email, rather than just opening the inbox.

Email deep links are useful if you want to save a link to a specific emailâ€”for example, if you want to add the link to your personal calendar.

Outlook doesn't natively support creating deep links to emails (unless you use a workaround with the To Do app). This app provides an alternative solution.

How Do I Use This?

  1. Go to your Outlook inbox.
  2. Open the Chrome DevTools Console (press F12).
  3. Go to the Network tab.
  4. In the search bar, search for service.svc?action=GetItem.
  5. Click the email you want to get a deep link for.
  6. A new item should appear at the bottom of the network traffic. Click it, then click Response.
  7. Copy and paste the entire response into the text input above.
  8. The app automatically generates a deep link to the email, which you can copy/paste elsewhere.
